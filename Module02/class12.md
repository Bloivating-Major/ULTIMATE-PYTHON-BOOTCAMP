# Basic Git Setup

## Configuration

- Global User Config
    - git config --global user.name "Sambhav"

- Global Email Config
    - git config --global user.email "sambhav12wakhariya@gmail.com"

- Core Editor Config
    - git config --global core.editor "code --wait"

- Auto CRLF
    - git config --global core.autocrlf
